## 当前提示词原则

提示词已强化为：

- 知识库是主要参考标准
- 回复必须跟随客户来信语种
- 中文解释只能放在 `translationZh` 或 `internalSuggestion`
- 不编造订单状态、物流单号、仓库核实、附件、退款、补发、折扣、赔偿、法律结论或平台处理结果
- 中高风险或 `requireHumanReview=true` 的知识库条目必须人工审核
- 出现占位符、未核实运单号、未上传附件却引用附件、承认责任或利益承诺时必须阻断自动发送
